# 🕊️ NayePankh Foundation — Impact Website

A modern, multi-page website built for NayePankh Foundation 
as part of the Technical Internship Task.

🔗 **Live Demo:** [Click Here](https://ajaymishraj.github.io/NayePankh-website-task)

---

## 📌 About the Project

NayePankh Foundation's current website lacks several important 
features. This project reimagines it with a fully responsive, 
animated, AI-powered multi-page website that better represents 
their mission and impact.

---

## ✨ Features

- 🌗 Dark / Light Mode Toggle
- 📊 Animated Impact Counter (live counting stats)
- 🤖 AI Chatbot (answers questions about NayePankh)
- 🧠 Volunteer Role Finder Quiz (AI recommends best role)
- 💸 Donation Impact Calculator
- 📱 Fully Mobile Responsive
- 🎞️ Scroll Animations throughout
- 📄 5 Complete Pages

---

## 📄 Pages

| Page | Description |
|------|-------------|
| Home | Hero section, animated stats, mission |
| About | Foundation story, team, timeline |
| Volunteer | Role finder quiz + AI chatbot |
| Stories | Impact stories from beneficiaries |
| Donate | Donation calculator + donate button |

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- Vanilla JavaScript
- Claude AI API (chatbot + role finder)
